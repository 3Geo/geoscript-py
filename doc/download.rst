Downloads
=========

Stable
------

The latest stable release of GeoScript PY is 1.0. Two distributions are available for download:

  .. cssclass:: external  

      `Full <http://github.com/downloads/jdeolive/geoscript-py/geoscript-1.0.tar.gz>`_ 
         Contains GeoScript sources and required GeoTools libraries

  .. cssclass:: external  

      `Source <http://github.com/downloads/jdeolive/geoscript-py/geoscript-1.0-src.tar.gz>`_ 

         Contains only GeoScript sources

Experimental
------------

.. warning::

   Experimental releases are unstable in nature and contain api that is subject to change without deprecation.
